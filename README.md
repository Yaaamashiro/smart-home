# 開発環境

- Raspberry Pi Zero 2 W
- Ubuntu Server 24.04.2 LTS
- Python 3.12.10
- Django 5.2.1
- PostgreSQL 17

# Home の初期設定

```
curl -o setup.sh https://raw.githubusercontent.com/Yaaamashiro/smart-home/refs/heads/main/home/setup.sh
bash setup.sh
```
